### Technology layers

ZkNoid offers the following reusable modules:

- GameHub (recursive + proof store + leaderboards)
- Random (pseudorandom numbers generator)
- Matchmaking (makes opponents to find each other)
- Lobby Manager (create lobby and play with friends)
- Session Keys (do not trigger wallet popup on every transactions)
