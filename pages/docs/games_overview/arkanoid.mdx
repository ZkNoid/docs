## Arkanoid

### Description

In Arkanoid, your objective is to break all the bricks on the screen using a bouncing ball and a platform. 
You can control the game by using the left and right arrow keys on your keyboard to move the platform. 
You need to bounce the ball and prevent it from falling off the bottom of the screen.


<Image src={'/games_overview/arkanoid.png'} width={1600} height={1077}/>

**Network layer:** protokit appchain

### Provable part

In the Arkanoid game, on every game tick the game state is reproduced inside the circuit. 
While playing, the recursive zero knowledge proof is accumulated verifying the current game score correctness.
At the end of each game round, scores are added to the leaderboard accompanied by proof of their fairness
